SAP Beta Deck Manager

This website allows you to save and load decks in the Super Auto Pets test server.

To use it enter your login info for the SAP test (you need a seperate account for the test server, it is seperate from the live version). Once you login, a list of your decks should appear. Click the Copy to Clipboard button to copy the deck code. To load a deck, paste the deck code into the box at the bottom, and click Add Deck.

All requests are sent directly from your device to the Super Auto Pets server; using this site does not give me access to your account.
